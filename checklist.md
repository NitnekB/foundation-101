# The website checklist

## Dev

* [ ] GitHub Hosting
* [ ] Custom Domain name
* [ ] Multiple pages website
* [ ] Compiled CSS
* [ ] Compiled JavaScript
* [ ] Use a form endpoint API
* [ ] Check error on form **before submit**

## Design

* [ ] Nice palette color
* [ ] Typography lover with a good font pairing
* [ ] Responsive website
* [ ] Favicon

## Seo

* [ ] Robots.txt file
* [ ] Sitemap.xml
* [ ] Title / Description / H1 / url different for every pages
